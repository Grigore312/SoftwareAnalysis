To run this project, first open dataset_creation.ipynb, and put your own github token. Then run all cells. This will create a csv file with all 50 repositories

Then, run dataset_preprocessing.ipynb. This will create a new csv file with the 12 repositories.

The rest of the files are independent of each other, but all require the csv file generated by dataset_preprocessing.ipynb. These files contain the tests required for the three research questions.
